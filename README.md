# FindDoctor
> FindDoctor 為蒐尋醫生評價網頁，希望能減少病患求醫焦慮、讓更多人看見好醫師。

_**Link :**_ https://finddoctor.abinitio.fun/

使用說明：選擇臺北榮總科別及醫生選單或直接輸入醫生姓名後，可取得榮總感謝函、Blog、Ptt、Dcard、Google Review、良醫網及司法院資訊。
